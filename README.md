# GemmaTokenizerExtractor

## 初始化專案

1. 建立專案

    ```bat
    mkdir GemmaTokenizerExtractor
    cd GemmaTokenizerExtractor

    python -m venv .venv
    .venv\Scripts\activate.bat

    git ignore python > .gitignore

    pip install transformers torch sentencepiece
    ```

2. 到 <https://huggingface.co/google/gemma-2-9b-it/tree/main> 下載所有 `*.json` 檔案，放到 `./gemma-2-9b-it` 資料夾下。

    ```sh
    python extract_gemma_tokens.py
    ```

3. 建立版控

    ```sh
    git init
    git add .
    git commit -m "Initial commit"
    ```
